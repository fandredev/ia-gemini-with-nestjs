## Clone this repo

```
git clone git@github.com:fandredev/ia-gemini-with-nestjs.git
```

## Go to directory

```
 cd ia-gemini-with-nestjs
```

## Install dependencies

```
 yarn
```

## Run server

```
 yarn run dev
```

## Run tests

```
 yarn run test
```

## Run coverage

```
 yarn run test:cov
```

## Run linter

```
 yarn run format
```
